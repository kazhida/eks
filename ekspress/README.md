紆余曲折あって、もとのExpress.jsとは、全く別物になった Web Application Framework。

Express.jsのミドルウェアという考え方をもとに、Koa.jsのコンテクストという導入した。
コンテクストは、不変オブジェクトで、ミドルウェアは、渡されたコンテクストから、新しいコンテクストを返す。
その循環によって、アプリケーションを構築する。
